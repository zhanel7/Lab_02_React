# Лабораторная работа 2.1

## Как запустить проект:

1. Откройте терминал в папке task1
2. Установите зависимости: `npm install`
3. Запустите проект: `npm start`
4. Откройте браузер по адресу: http://localhost:3000

## Компоненты:

1. **FragmentLayout** - демонстрация использования фрагментов
2. **ItemList** - отображение списка с помощью .map()
3. **CombinedFragmentList** - комбинированный компонент

## Задания:
- ✓ Фрагменты без лишних оберток
- ✓ Маппинг массива в JSX
- ✓ Использование ключей (key)
- ✓ JSX выражения для динамических значений
