# Лабораторная работа 2.2

## Как запустить проект:

1. Перейдите в папку task2
2. Установите зависимости: `npm install`
3. Запустите: `npm start`
4. Откройте: http://localhost:3001 (может потребоваться изменить порт)

## Компоненты:

1. **Card** - переиспользуемый компонент карточки
2. **ProductList** - список продуктов с маппингом
3. **Section** - компонент секции с фрагментами

## Особенности:
- ✓ Динамические пропсы
- ✓ Композиция компонентов
- ✓ Фрагменты для группировки
- ✓ Правильные ключи для списков
